프로젝트의 코드 품질을 종합적으로 검사합니다. 아래 4단계를 순서대로 실행하고, 각 단계의 결과를 표로 요약해주세요.

## 실행 순서

**1단계 — 타입 검사**
```
npx tsc --noEmit
```
오류가 없으면 ✅, 있으면 ❌ 와 함께 오류 목록을 출력합니다.

**2단계 — 린트 검사**
```
npm run lint
```
경고/오류가 없으면 ✅, 있으면 ❌ 와 함께 항목을 출력합니다.

**3단계 — 빌드 확인**
```
npm run build
```
성공하면 ✅, 실패하면 ❌ 와 함께 빌드 오류를 출력합니다.

**4단계 — 테스트 실행**
```
npx playwright test
```
전체 통과하면 ✅, 실패가 있으면 ❌ 와 함께 실패한 테스트 이름을 출력합니다.

## 최종 요약

모든 단계가 끝나면 아래 형식으로 결과를 요약합니다:

| 항목 | 결과 | 비고 |
|------|------|------|

| 린트 | ✅/❌ | |
| 빌드 | ✅/❌ | |
| 테스트 | ✅/❌ | |
